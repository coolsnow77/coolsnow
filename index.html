JQ [jq: Linux下json的命令行工具]

轻量级的弹性的命令行JSON 处理器.

使用JQ 在shell 中处理JSON

官方网址：http://stedolan.github.io/ 官方网址: https://atom.io/ [Modern,hackable, text editor]

Download

jq-1.4.tar.gz

Install

      Mac OS:
              sudo brew install jq
      Linux :
          tar -zxvf jq-1.4.tar.gz
          ./configure && make && make install
          ln -s /usr/local/bin/jq /usr/bin/jq;

example

file_jq.json 下载 file_jq.json 
          1. 格式化输出
          ^_^coolsnow77:myJQ @$jq '.' file_jq.json
            {
              "Archiver-Version": "Plexus Archiver",
              "Build-Id": "",
              "Build-Jdk": "1.7.0_07",
              "Build-Number": "",
              "Build-Tag": "",
              "Built-By": "cporter",
              "Created-By": "Apache Maven",
              "Implementation-Title": "northstar",
              "Implementation-Vendor-Id": "com.test.testPack",
              "Implementation-Version": "testBox",
              "Manifest-Version": "1.0",
              "appname": "testApp",
              "build-date": "02-03-2014-13:41",
              "version": "testBox",
              "myname": {
                "sex": "male",
                "age": 30,
                "hobby": "book"
              },
              "kyname": {
                "sex": "female",
                "age": 32,
                "hobby": "jj"
              }
            }

          2. 获取所有的keys:
          ^_^coolsnow77:myJQ @$jq 'keys' file_jq.json
            [
              "Archiver-Version",
              "Build-Id",
              "Build-Jdk",
              "Build-Number",
              "Build-Tag",
              "Built-By",
              "Created-By",
              "Implementation-Title",
              "Implementation-Vendor-Id",
              "Implementation-Version",
              "Manifest-Version",
              "appname",
              "build-date",
              "kyname",
              "myname",
              "version"
            ]
          3.1 组合获取：(获取所有的sex，age)
          ^_^coolsnow77:myJQ @$jq  '.["myname", "kyname"] | {"sex", "age"}' file_jq.json
            {
              "sex": "male",
              "age": 30
            }
            {
              "sex": "female",
              "age": 32
            }

          3.2 合并为一行输出, 添加 -c 参数
          ^_^coolsnow77:myJQ @$jq -c '.["myname", "kyname"] | {"sex", "age"}' file_jq.json
            {"sex":"male","age":30}
            {"sex":"female","age":32}
